# SeedBruteForce

This project is designed for brute-forcing seed phrases to recover lost cryptocurrencies. The goal is to demonstrate a brute-force approach for educational purposes only.

## Features
- **Brute-force attack** on seed phrases (12-24 words).
- **Written in Python** for simplicity and flexibility.
- Educational demonstration of seed phrase recovery techniques.

## Getting Started

### Prerequisites
- **Python 3.x** must be installed on your computer. You can download it from [here](https://www.python.org/downloads/).

### Installation
1. Clone this repository to your local machine:
    ```bash
    git clone https://github.com/jaallaal/seedbruteforce.git
    ```
2. Navigate into the project folder:
    ```bash
    cd seedbruteforce
    ```
3. Install the required dependencies (if any):
    ```bash
    pip install -r requirements.txt
    ```

### Usage
Run the brute-force script by using the following command:
```bash
python brute_force.py
